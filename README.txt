## How to publish
1. Create the repo named `<your-username>.github.io` (public).
2. Upload **all files/folders in this folder** to the root of the repo.
3. In **Settings → Pages**, set:
   - Source: *Deploy from a branch*
   - Branch: `main` / folder: `/ (root)`
4. Visit `https://<your-username>.github.io/`.

> Replace placeholder images/videos and links as you like.
